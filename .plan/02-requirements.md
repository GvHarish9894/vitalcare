# Functional Requirements

## Authentication
- Login
- Register
- Forgot Password
- Logout

## Dashboard
- Today's summary
- Latest reading
- Pending sync status
- Quick actions

## Record Vitals
Validation:
- SpO₂: 70-100
- Heart Rate: 20-250
- BP Systolic: 50-250
- BP Diastolic: 30-180

Save Flow:
1. Validate
2. Save Room
3. Update UI
4. Queue Sync

## History
- Group by date
- Search
- Filter Today/Week/Month

## Analytics
Daily, Weekly, Monthly charts.
